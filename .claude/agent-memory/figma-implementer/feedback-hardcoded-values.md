type: feedback
---

토큰에 없는 고정 px 값(56px, 44px, 18px 등)은 JSX style 속성에서 숫자 리터럴(height: 56) 대신 문자열('height: "56px"') 형태로 작성하고, `/* ⚠️ --spacing-XX 토큰 없음 */` 주석을 함께 남긴다.

**Why:** 숫자 리터럴은 px 단위가 암묵적으로 적용되지만, 문자열 표기가 명시적이며 토큰 미등록 의도를 코드에서 확인할 수 있다. 2026-05-13 감사에서 이 방침으로 수정됨.

**How to apply:** 새 컴포넌트 구현 시 --spacing-\* 토큰에 없는 px 값이 필요하면 무조건 문자열 px + ⚠️ 주석 패턴을 사용할 것. 토큰을 임의로 추가하거나 가장 가까운 값으로 반올림하지 말 것.
